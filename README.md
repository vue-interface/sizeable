# Sizeable

A Vue mixin that adds the size class to the component.

## Installation

NPM

    npm i @vue-interface/sizeable

Yarn

    yard add <%= pkg.name %>

## Props

### size

The size of the form control. Possible values: `sm`, `md`, `lg`.

@default  'md'
@property  String


## Computed

### sizeableClass

The sizeable classes with the appropriate prefix.

@return  String
